# Article Or Announcement

Use this skill for outward-facing writing that is not core README maintenance.

## Rules

- Keep technical claims aligned with the current README and implementation.
- Avoid promising model hosting, weights, or platform support that this repo does
  not provide.
- For English replies, keep wording natural and concise; do not over-explain the
  internal development process.
- For Hugging Face / ModelScope / article drafts, distinguish this custom node
  from downstream models such as Wan/VACE.
